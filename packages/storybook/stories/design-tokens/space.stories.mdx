# Space

Use the design tokens for space only for space within (inline/block) or between components (column/row), not to define the size of elements.

## Inline spacing

[Figma - Utrecht Design System - Space Inline](https://www.figma.com/file/msb3CfQBefPoruqNQ968Zh/Utrecht-Design-System?node-id=2%3A634)

Inside components, use inline spacing before and after text. In CSS you can use the following properties:

- `margin-inline-start`
- `margin-inline-end`
- `padding-inline-start`
- `padding-inline-end`

## Block spacing

[Figma - Utrecht Design System - Space Block](https://www.figma.com/file/msb3CfQBefPoruqNQ968Zh/Utrecht-Design-System?node-id=1%3A1101)

Inside components, use block spacing above and below text. In CSS you can use the following properties:

- `margin-block-start`
- `margin-block-end`
- `padding-block-start`
- `padding-block-end`

<SpaceTokensTable orientation="block" tokens={Object.values(designTokens["utrecht"]["space"]["block"])} />

Ruimte tussen blokken is handig voor touch screen gebruikers, om te scrollen zonder onbedoeld iets te activeren. Gebruikers met een bevende hand kunnen de site lastiger gebruiken als grote delen van het scherm een button of link zijn. Het kan bijvoorbeeld gebruiksvriendelijk zijn om 44px ruimte te houdden tussen blokken, de minimale _target size_ afmeting uit [WCAG eis 2.5.5](https://www.w3.org/TR/WCAG21/#target-size).

## Distance between components: columns

To create distance horizontally between components, you can use the following tokens for the following properties:

- `margin-inline-start`
- `margin-inline-end`
- `padding-inline-start`
- `padding-inline-end`
