# PDF

Data List moet opgebouwd worden met de volgende tags: `<L>`, `<LI>`, `<Lbl>` en `<LBody>`.

```text
<L> {
  <LI> {
    <Lbl> {
        "Voornaam"
    }
    <LBody> {
        "Bobby"
    }
  }
  <LI> {
    <Lbl> {
        "Achternaam"
    }
    <LBody> {
        "Tables"
    }
  }
}
```
