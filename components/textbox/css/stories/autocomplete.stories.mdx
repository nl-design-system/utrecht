# Textbox met autocomplete

Voor [WCAG eis 1.3.5](https://www.w3.org/TR/WCAG21/#identify-input-purpose) moet je altijd een `autocomplete` waarde gebruiken als een toepasselijke waarde beschikbaar is. Op deze pagina is een overzicht van veelvoorkomende velden die een `autocomplete` attribuut moeten gebruiken.
